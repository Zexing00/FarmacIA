# FarmacIA
Trabajo de Ingenieria de Software 3ºIngenieria Informatica
Introducción:
Programa con sistema de usuarios de dos niveles, usuarios corrientes y administradores, dentro del programa el usuario podrá registrar los medicamentos que necesite así como guardar las fechas de caducidad de los medicamentos que toma y llevar un control del consumo aprox semana tras semana. Además el usuario también podrá consultar el prospecto de cualquier medicamento y junto con una IA consultar dudas que pueda tener. El administrador podrá acceder a la información del resto de usuarios.  

Pros:
Uso de datos externos, puntúa muy bien en la práctica.
Tendríamos cubiertas las tres bases de datos que se nos piden.
Conseguir los datos no será muy complicado, son públicos todos los prospectos. 

Funciones:
Guardar prospectos de los medicamentos y poder mirarlos en cualquier momento.
