## Build overview (mas especifico)
- We are building a mobile application consisting on a digital "farmacy". in this app, there will be a list, composed of various medications and their prospects, which will be stored in an SQL database. The app also has a login system, with user name and password also store in a database, there will be two types of users, user and administrator. The main gimmick of this application will be a digital pill box, where you can select the medications you are currently taking and keep trace of them. 

## Restrictions
- Use of ORMs is forbidden
- Must use DAOs for database access
- Add libraries when needed
- Programming language will be Java.
